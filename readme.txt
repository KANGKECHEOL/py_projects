- 최초1회만
  만약, 프로젝트를 먼저 작업하다가, git를 사용한 경우
  1) git clone을 다른 위치에서 수행
  2) clone을 통해 만들어진 위치에 먼저 만들어진 프로젝트 내용을 이동
  3) vs code 에서 커밋(commit) and push 작업을 수행
  4) 단, 최초수행시 git 오류가 날 수 있고 이경우 터미널에서
     git config --global user.name "kangkicheol" 
     git config --global user.email "meem0428@kmou.ac.kr"
     명령을 1회 수행해주 (현재 프로젝트 위치에서) 처린된다.
  5) 앞으론느 clone 한 프로젝트 위치가 실제 작업할 곳이 된다.
